# 问题分析

## 错误：SQL 语法错误
- **错误信息**：`You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near 'groups g\n        WHERE EXISTS (\n            SELECT 1 FROM group_members gm2 WHER' at line 26`
- **原因**：`groups` 是 MySQL 的保留字，在 SQL 查询中需要使用反引号包围
- **具体问题**：在 `groups.py` 文件的 `list_groups` 函数中，SQL 查询语句中的 `groups` 表名没有使用反引号包围

# 修复方案

## 修改文件：`app/api/v1/endpoints/groups.py`

### 修改点 1：`list_sql` 查询语句
- **修改前**：`FROM groups g`
- **修改后**：`FROM `groups` g`

### 修改点 2：`count_sql` 查询语句
- **修改前**：`FROM groups g`
- **修改后**：`FROM `groups` g`
